# armour

## Development

### Prerequisites

* [`gleam`](https://gleam.run/getting-started/installing/)
* [`npm`](https://github.com/Schniz/fnm) (link is for FNM)

### Dev server

```shell
gleam run -m lustre/dev start
```

Then go to http://localhost:1234

Code goes in `src`.


### Build

```shell
npm run make
```
